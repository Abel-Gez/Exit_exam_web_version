# Exit Exam Prep Web App

A Next.js web application for practicing exit exams, built with TypeScript, Firebase, and Tailwind CSS.

## Features

- User authentication (Email/Password, Google)
- Department-based exam navigation
- Interactive quiz interface
- Study notes with rich text editing
- User profile and subscription management
- Responsive design for all devices

## Prerequisites

- Node.js 16.8 or later
- npm or yarn
- Firebase project with Firestore and Authentication enabled

## Setup
